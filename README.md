# ETF 智能投研系统 v0.1.0 发布说明

发布日期：2026-02-12

## 版本概览

这是第一个可用发布版本，已具备“真实数据驱动 + 前端中文化 + Windows 本地部署”完整链路。

## 主要新增

- 前端核心页面全部打通：组合总览、策略回测、基金筛选、策略参数、定投模拟、数据查询
- 回测页支持：样本外验证、参数敏感性分析、结果导出
- 筛选页支持：排序、导出、候选池联动、无结果推荐参数与一键重试
- 策略页支持：参数在线编辑、参数校验提示、候选池一键应用
- 定投页支持：固定金额/固定份额、分红方式入口、分笔记录中文日期展示
- 首页支持：风控状态、系统状态、最近交易日志、刷新数据

## 真实数据能力

- Web 接口已接入真实数据链路（本地 CSV 优先，缺失时实时拉取）
- 支持主备数据源、重试与数据标准化
- 前端显示数据来源（实时数据/本地历史数据/模拟数据）

## 中文化与交互体验

- 用户可见文案统一为中文
- 关键操作统一三态反馈：加载中 / 成功 / 失败
- 交易信号与交易原因支持中文展示

## 文档

- Windows 用户手册：`docs/用户使用手册-Windows.md`

## 质量与验证

- 全量测试通过：`pytest -q`（当前 204 通过）
- Web 接口与页面测试通过：`pytest tests/test_web/test_app.py -q`

## 已知事项

- 外部数据源可能偶发 `RemoteDisconnected`，系统会尝试重试或切换备用源
- 某些严格筛选条件可能无结果，页面会给出推荐参数并支持一键重试

## 升级建议

- Windows 部署用户升级后建议先执行：

```powershell
pytest tests/test_web/test_app.py -q
pytest -q
```
